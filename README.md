# recopp
** work in progress **

reconstruction library in C++

## Dependencies

* ROOT
* ROOTANA
* OpenCV

Before compiling, set the variables `ROOTANASYS` and `OPENCVSYS` in your environment.

## Installation

`git clone https://github.com/piacent/recopp.git`

`cd recopp`

`mkdir build-dir`

`cmake ..`

`cmake --build .`
